# PX4-UNITY-Simulator
Unity3d Environment for UAV simulation, with connection to **PX4-SITL**

## Platform
**[HARDWARE]**
- Ubuntu 18.04
- Intel® Xeon(R) CPU E3-1535M v6 @ 3.10GHz × 8 
- NVIDIA Quadro M2200

**[SOFTWARE]**
- Unity 2020.3.6f1

**[NOT SUPPORTED ON UNITY-UBUNTU]**
- [HDRP] is not supported
- [Substance engine] is not supported
